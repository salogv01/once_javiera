# ¿Cómo crear un repositorio?

## Pasos

- 1: Abrir tu cuenta de GitHub.
- Paso 2: Entrar a tu perfi.
- Paso 3: Ir a la pestaña de repositorios y entrar.
- Paso 4: Darle al botón verde que está arriba a la derecha que dice **"New"**.
- Paso 5: Ponerle nombre al repositorio y ***asegurarse de que sea público.***.
- Paso 6 Darle a "Crear".
- Fin.
