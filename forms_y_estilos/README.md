# Formularios y estilos.
En la clase del día de hoyr, hicimos una actividad sobre formularios, después de leer todo un documento, empezamos con la practica, cogiendo un documento HTML y CSS para modificar distintas cosas que el profesor nos indico que cambiaramos.

### En mi caso, lo que hice en HTML fue lo siguiente.

-Empece abriendo el documento en Visual Studio y copiando lo que el profe puso en el documento para pegarlo. Uno, el HTML, y el otro, el CSS.

-Después de mirar lo que había en dicho documento, empece agregando una sección llamada "Estado civil", poniendo lo siguiente:

💗 `label` para que el texto reconociera lo que estaba copiando.
    
💗 `For="Estado¨` es para que conecte con la etiqueta del mismo `id="estado"`

💗 El texto `Estado civil` es lo que vera el usuario.

💗 El `<span class="obligatorio">*</span>` añade el asterisco para indicar que es obligatorio.

💗 `type="text` indica que es un campo de texto.

💗 `id="estado` y `name="estado"` identifican el campo; el name es lo que se envía al servidor.

💗 `placeholder="Cuentanos tu historia"` muestra un texto dentro del campo hasta que el usuario empiece a escribir.

💗 `maxlength="100.000"` determina el número máximo de caracteres permitidos.
